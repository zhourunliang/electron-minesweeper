### 基于electron的扫雷游戏
+ electron
### 本地环境配置
+ 安装node.js
+ npm install -g electron
+ npm install -g electron-forge
+ git clone https://github.com/zhourunliang/electron-minesweeper
+ npm install 
+ electron-forge start
